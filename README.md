# 🚢 타이타닉 생존자 예측

## 프로젝트 소개
타이타닉 승객 데이터를 분석하고
AI 모델로 승객의 생존 여부를 예측하는 프로젝트입니다.

## 사용 기술
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Random Forest

## 모델 정확도
- 83.24%

## 주요 분석 결과
- 여성 생존율 74% vs 남성 생존율 19%
- 1등석 생존율 63% vs 3등석 생존율 24%
- 이름의 호칭(Mr, Mrs, Miss)이 생존율에 영향

## 데이터
- Kaggle Titanic Dataset
- 훈련 데이터 891명 / 테스트 데이터 418명
